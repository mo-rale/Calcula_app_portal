Constant Rule

If f(x) = c, then f'(x) = 0

The derivative of a constant function is always zero. This is because a constant function represents a horizontal line, which has a slope of zero at every point. No change means zero rate of change!

Examples:

If f(x) = 5, then f'(x) = 0

If g(x) = -100, then g'(x) = 0

If h(x) = π, then h'(x) = 0

Power Rule

If f(x) = xn, then f'(x) = n·xn-1

The power rule is one of the most fundamental rules in calculus. When you have a variable raised to a power, multiply by the exponent and then decrease the exponent by 1. This works for any real number exponent.

Examples:

If f(x) = x3, then f'(x) = 3x2

If g(x) = x10, then g'(x) = 10x9

If h(x) = √x = x1/2, then h'(x) = (1/2)x-1/2

Identity Rule

If f(x) = x, then f'(x) = 1

This is a special case of the power rule where the exponent is 1. The function f(x) = x is a straight line with a slope of 1, so its derivative is always 1. Simple yet powerful!

Examples:

If f(x) = x, then f'(x) = 1

Using power rule: f(x) = x1, f'(x) = 1·x0 = 1·1 = 1
